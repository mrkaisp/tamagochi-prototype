## キャラクター分岐影響テーブル

### フェーズ2（芽→茎）分岐への影響

各キャラクターの遺伝因子がフェーズ2分岐スコアに与える影響：

| 遺伝因子 | フェーズ2分岐スコアへの加点 | 説明 |
|---------|------------------------|------|
| 陽 | +5 | しなる寄りに成長しやすい |
| 陰 | +0 | 中立、他の要素に依存 |

**計算方法**:
- 単一遺伝因子: 上記の加点値をそのまま使用

### フェーズ3（茎→蕾）分岐への影響

各キャラクターの遺伝因子がフェーズ3形状スコアに与える影響：

| 遺伝因子 | フェーズ3形状スコアへの加点 | 説明 |
|---------|------------------------|------|
| 陽 | +10 | 大輪・まるまる寄りに成長しやすい |
| 陰 | +5 | ひらひら・ちいさめ寄りに成長しやすい |

**計算方法**:
- 単一遺伝因子: 上記の加点値をそのまま使用

### 性格の影響

各キャラクターの性格（芽性格）は成長分岐には直接影響しないが、メンタルパラメータの増減に影響する可能性がある（将来の拡張予定）。

| 性格 | 説明 |
|------|------|
| わんぱく | 明るく元気、メンタルが上がりやすい |
| やさしい | 穏やか、メンタルが安定しやすい |
| クール | 冷静、メンタルが下がりにくい |
| おっとり | のんびり、メンタルが安定 |
| きまぐれ | 気分屋、メンタルが変動しやすい |

## 実装上の注意事項

1. **遺伝因子の実装**: 現在の実装では単一遺伝因子（陰、陽）のみをサポート。

2. **キャラクター名の決定**: 
   - 種段階: 種タイプに基づいて決定（例: 陽→陽っち）
   - 花段階: 成長分岐の結果に基づいて決定（例: たんぽぽ、さくら、ひまわり）

3. **分岐スコア計算**: 陰陽の加点値を使用する。
