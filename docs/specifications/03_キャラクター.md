## キャラクター分岐影響テーブル

### フェーズ2（芽→茎）分岐への影響

各キャラクターの遺伝因子がフェーズ2分岐スコアに与える影響：

| 遺伝因子 | フェーズ2分岐スコアへの加点 | 説明 |
|---------|------------------------|------|
| 太陽 | +5 | まっすぐ寄りに成長しやすい |
| 月 | +0 | 中立、他の要素に依存 |
| 風 | +2 | しなる寄りに成長しやすい |
| 雨 | +2 | しなる寄りに成長しやすい |
| 月×風 | +1 | 複合遺伝因子（月+風の平均） |
| 雨×月 | +1 | 複合遺伝因子（雨+月の平均） |
| 太陽×風 | +3.5 | 複合遺伝因子（太陽+風の平均） |

**計算方法**:
- 単一遺伝因子: 上記の加点値をそのまま使用
- 複合遺伝因子: 各遺伝因子の加点値の平均を計算（例: 月×風 = (0+2)/2 = +1）

### フェーズ3（茎→蕾）分岐への影響

各キャラクターの遺伝因子がフェーズ3形状スコアに与える影響：

| 遺伝因子 | フェーズ3形状スコアへの加点 | 説明 |
|---------|------------------------|------|
| 太陽 | +10 | 大輪・まるまる寄りに成長しやすい |
| 月 | +5 | ひらひら・ちいさめ寄りに成長しやすい |
| 風 | +5 | ひらひら・ちいさめ寄りに成長しやすい |
| 雨 | +10 | まるまる寄りに成長しやすい |
| 月×風 | +5 | 複合遺伝因子（月+風の平均） |
| 雨×月 | +7.5 | 複合遺伝因子（雨+月の平均） |
| 太陽×風 | +7.5 | 複合遺伝因子（太陽+風の平均） |

**計算方法**:
- 単一遺伝因子: 上記の加点値をそのまま使用
- 複合遺伝因子: 各遺伝因子の加点値の平均を計算（例: 月×風 = (5+5)/2 = +5）

### 性格の影響

各キャラクターの性格（芽性格）は成長分岐には直接影響しないが、メンタルパラメータの増減に影響する可能性がある（将来の拡張予定）。

| 性格 | 説明 |
|------|------|
| わんぱく | 明るく元気、メンタルが上がりやすい |
| やさしい | 穏やか、メンタルが安定しやすい |
| クール | 冷静、メンタルが下がりにくい |
| おっとり | のんびり、メンタルが安定 |
| きまぐれ | 気分屋、メンタルが変動しやすい |

## 実装上の注意事項

1. **複合遺伝因子の実装**: 現在の実装では単一遺伝因子（太陽、月、風、雨）のみをサポート。複合遺伝因子（月×風、雨×月、太陽×風）は将来の拡張として実装予定。

2. **キャラクター名の決定**: 
   - 種段階: 種タイプに基づいて決定（例: 太陽→たんぽっち）
   - 花段階: 成長分岐の結果に基づいて決定（例: たんぽぽ、さくら、ひまわり）

3. **分岐スコア計算**: 複合遺伝因子の場合は、各遺伝因子の加点値の平均を計算する。
