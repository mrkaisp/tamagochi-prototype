## 成長分岐表（集約）

### フェーズ1（種→芽）

| 出力 | 主条件 |
|---|---|
| ハート芽 | 光>=50、陽寄り |
| 棘芽 | 光<50、陰寄り |

### フェーズ2（芽→茎）

| 出力 | 主条件 |
|---|---|
| まっすぐ | 総合70〜100（良好） |
| しなる | 総合40〜69（中庸） |
| つる | 総合0〜39（低調） |
| ふつう | その他 |

| 出力 | 主条件 |
