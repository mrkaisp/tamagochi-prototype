## フェーズ2（芽→茎）

### 条件A：パラメータ

- 栄養（water_level）: 0〜100
- 光（light_level）: 0〜100（成長時に40以上必要）
- 精神健康状態（mental_level）: 70〜100で良好

### 分岐例

| 茎タイプ | 条件（概略） |
|---|---|
| しなる（雑草タイプ） | 総合スコア50〜100（栄養+光+メンタルの平均+種バイアス+メンタル高値バイアス） |
| つる（ひね曲がり） | 総合スコア0〜49 |
| ふつう | その他 |

### 実装詳細

- **成長条件**: 光レベルが40以上で芽→茎に成長
- **総合スコア計算**: (water_level + light_level + mental_level) / 3 + 種バイアス（環境整備機能削除によりenvironment_levelは使用停止）
- **種バイアス**: 陽+5、陰0
- **メンタル高値バイアス**: メンタル70以上で+5
- **光レベルのリセット**: 成長後、光レベルは0にリセットされる

> 注: 属性（種、芽、棘/ハート）で重み付けを行い最終決定。
