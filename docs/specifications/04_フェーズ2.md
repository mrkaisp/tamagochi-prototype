## フェーズ2（芽→茎）

### 条件A：パラメータ

- 栄養（water_level）: 0〜100
- 環境（environment_level）: 0〜100
- 光（light_level）: 0〜100（成長時に40以上必要）
- 精神健康状態（mental_level）: 70〜100で良好

### 分岐例

| 茎タイプ | 条件（概略） |
|---|---|
| まっすぐ（誠実・王道） | 総合スコア70〜100（栄養+光+環境+メンタルの平均+種バイアス） |
| しなる（雑草タイプ） | 総合スコア40〜69 |
| つる（ひね曲がり） | 総合スコア0〜39 |
| ふつう | その他 |

### 実装詳細

- **成長条件**: 光レベルが40以上で芽→茎に成長
- **総合スコア計算**: (water_level + light_level + environment_level + mental_level) / 4 + 種バイアス
- **種バイアス**: 太陽+5、月0、風+2、雨+2
- **メンタル高値バイアス**: メンタル70以上で+5
- **光レベルのリセット**: 成長後、光レベルは0にリセットされる

> 注: 属性（種、芽、棘/ハート）で重み付けを行い最終決定。
